# 召喚寵物戰鬥訓練師

手機優先的純前端網頁遊戲。玩家在 10x10 地圖上移動、收集資源、培養寵物，擊敗 3 位 Boss。

## 主介面

- 冒險地圖：主畫面，顯示 10x10 地圖與移動範圍。
- 人物資料：人物等級、EXP、AP、金幣、隊伍、威脅、道具、位置。
- 寵物介面：寵物資料、訓練、回血、復活。
- 道具介面：使用藥草、守護符、捕網、呼哨。
- 商店介面：購買補給道具。

## 移動規則

- 不使用骰子。
- 每次可選擇 10 格內任意格移動。
- 每次移動消耗 1 AP。
- Lv.1 人物 AP 上限為 60。
- AP 每 3 分鐘回復 1 點。
- 人物升級時 AP 會全滿。

## 讀秒回合制戰鬥

- 一次最多 5 隻我方寵物上場。
- 寵物依速度累積行動條。
- 我方寵物行動條滿時，玩家選擇技能。
- 敵方寵物行動條滿時自動攻擊。
- 技能分為攻擊、輔助、防守、治療。

## 成長與存檔

- 戰鬥勝敗都會獲得寵物 EXP、人物 EXP 與金幣。
- 打贏敵人有機率掉落新寵物。
- Boss 必定掉落寵物。
- 使用 `localStorage` 自動保存寵物、人物、金幣、AP、地圖進度與道具。
- 不需要帳號、不需要後端。
