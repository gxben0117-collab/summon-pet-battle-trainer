# 召喚寵物戰鬥訓練師

一個純前端網頁小遊戲。玩家扮演召喚寵物戰鬥訓練師，在 9x9 戰旗地圖上丟骰子，骰出點數後可以點選可移動範圍內的格子前進。玩家移動後，地圖上的 10 位對手會各自往玩家靠近一格；踩到對手、被追上，或抵達右上角 Boss 格，都會進入寵物戰鬥。

寵物有等級、力量、血量、速度、經驗值，以及五行屬性。五行相剋為：木克土、土克水、水克火、火克金、金克木。攻擊方克制防守方時，傷害會有 1.2 倍加成。

戰鬥採全隊自動戰鬥。所有未陣亡寵物會一起上場，依速度排序自動攻擊，直到敵方全滅或我方全隊陣亡。沒有訓練師 3 滴血，也不能復活寵物；如果全部寵物陣亡，就遊戲失敗。

戰鬥打贏或打輸都會得到經驗值與金幣；打贏對手有機率掉落新寵物。金幣可在地圖商店購買未陣亡寵物回血，或訓練未陣亡寵物升級。

## 開始遊玩

直接用瀏覽器開啟 `index.html`。

## 檔案

- `index.html`：遊戲畫面結構
- `styles.css`：畫面樣式
- `app.js`：遊戲流程與判定邏輯
