# 召喚寵物戰鬥訓練師

純前端網頁遊戲。玩家扮演召喚寵物戰鬥訓練師，在 10x10 戰旗地圖上移動、收集道具、尋求村民幫助，並擊敗 3 位守關 Boss。

## 核心規則

- 地圖為 10x10。
- 地圖上有 20 位追殺敵人、3 位 Boss、村民與道具格。
- 玩家丟骰子後，可選擇骰點範圍內的任意格移動。
- 玩家移動後，敵方會往玩家靠近一格。
- 踩到敵人、被追上，或踩到 Boss 格，會進入全隊自動戰鬥。
- 擊敗 3 位 Boss 即通關。
- 全部寵物陣亡且無法復活時遊戲失敗。

## 寵物戰鬥

寵物有等級、力量、血量、速度、經驗值與五行屬性。戰鬥時所有未陣亡寵物自動上場，依速度排序攻擊，直到一方全滅。

五行相剋：

- 木克土
- 土克水
- 水克火
- 火克金
- 金克木

克制時傷害為 1.2 倍。

## 成長與資源

- 戰鬥勝敗都會獲得 EXP 與金幣。
- 打贏敵人有機率掉落新寵物。
- Boss 必定掉落寵物。
- 商店可購買回血、訓練、藥草、守護符。
- 陣亡寵物不能回血、不能訓練、不能復活。

## 道具與村民

道具：

- 藥草：回復全隊未陣亡寵物 HP。
- 守護符：跳過下一次敵方追殺回合。
- 捕網：移除近距離敵人。
- 呼哨：召喚一隻 Lv.1 隨機寵物。

村民會隨機提供回血、金幣、道具、經驗或幫忙甩開敵人。
