# 召喚寵物戰鬥訓練師

手機優先的純前端網頁遊戲。玩家在 10x10 地圖上丟骰移動，敵人會在玩家移動後追殺；目標是收集資源、培養寵物，擊敗 3 位 Boss。

## 手機介面

- 地圖是主畫面。
- 寵物、背包、商店、事件改成底部 Tabs。
- 文字改成短標籤與符號。
- 戰鬥紀錄只顯示短訊息，避免手機畫面塞滿文字。

## 地圖規則

- 10x10 地圖。
- 20 位追殺敵人、3 位 Boss、村民與道具格。
- 玩家丟骰後，可點選範圍內格子移動。
- 玩家移動後，敵方會往玩家靠近一格。
- 踩到敵人、被追上、踩到 Boss，都會進入戰鬥。

## 讀秒回合制戰鬥

- 一次最多 5 隻我方寵物上場。
- 寵物依速度累積行動條。
- 我方寵物行動條滿時，玩家選擇技能。
- 敵方寵物行動條滿時會自動攻擊。
- 全隊寵物陣亡且無法復活時，遊戲失敗。

## 技能類型

- 攻擊：一般攻擊、強擊。
- 輔助：加速全隊行動條。
- 防守：防守、全隊護盾。
- 治療：治療 HP 最低的隊友。

## 成長與存檔

- 戰鬥勝敗都會獲得 EXP 與金幣。
- 打贏敵人有機率掉落新寵物。
- Boss 必定掉落寵物。
- 使用 `localStorage` 自動保存寵物、金幣、地圖進度與道具。
- 不需要帳號、不需要後端。
