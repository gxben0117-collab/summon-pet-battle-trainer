# 召喚寵物戰鬥訓練師

一個純前端網頁小遊戲。玩家扮演召喚寵物戰鬥訓練師，在 9x9 戰旗地圖上丟骰子，骰出點數後可以點選可移動範圍內的格子前進。地圖會隨機生成 10 位對手，右上角有守關 Boss。踩到對手或 Boss 後會進入寵物戰鬥。

寵物有等級、力量、血量、速度、經驗值，以及五行屬性。五行相剋為：木克土、土克水、水克火、火克金、金克木。攻擊方克制防守方時，傷害會有 1.2 倍加成。

玩家共有 3 滴訓練師血量。輸給一般對手扣 1 滴血，輸給 Boss 扣 3 滴血。戰鬥打贏或打輸都會得到經驗值與金幣；打贏對手有機率掉落新寵物。金幣可在地圖商店購買回血、復活寵物、訓練升級。

## 開始遊玩

直接用瀏覽器開啟 `index.html`。

## 檔案

- `index.html`：遊戲畫面結構
- `styles.css`：畫面樣式
- `app.js`：遊戲流程與判定邏輯
